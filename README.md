##Sales Invoice App
A simple React application for creating a sales invoice and managing purchased products.
#Overview
This project allows users to add products with their name, price, and quantity.
The items are displayed in a table where users can also remove products and see the total price of all items automatically calculated.
#Features
Add products through a form
Display products in a table
Delete items from the invoice
Automatic total price calculation
Toggle form visibility
#Technologies
React
useState Hook
Tailwind CSS
#Purpose
This project was built as a practice project to improve React state management and list handling.
